Workspace MCP Server

This is a Streamable HTTP server example that requires OAuth authorization (Bearer token).

API Endpoints:

  • /api/health - Health check
  • /mcp - Streamable HTTP connection endpoint (requires authorization)

Streamable HTTP Usage

        # Connect to SSE using the token
        curl -H "Authorization: Bearer BEARER_TOKEN" http://0.0.0.0:8000/mcp
